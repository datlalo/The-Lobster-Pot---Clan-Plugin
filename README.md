# The Lobster Pot RuneLite Plugin

A RuneLite companion plugin for LobsterPot clan members.

The plugin does not log in to a website, store credentials, or send tokens. Access is checked inside
the RuneLite client by reading the logged-in character's own clan data and allowing the plugin only
when the character is in the `LobsterPot` clan.

The plugin fetches a public read-only JSON feed generated by the Discord bot. That feed contains
broadcasts, upcoming events, and bot-managed member profile details such as rank, join date, points,
next-rank progress, and pending claims. Broadcasts and events are visible even before logging in;
member profile details are matched to the logged-in LobsterPot character. Because this is an
external request, RuneLite should be expected to show its third-party IP warning.

## Current behavior

- Checks the active character's RuneScape clan membership through RuneLite's clan APIs.
- Allows access when the character's own clan name matches `LobsterPot` after simple punctuation and
  spacing normalization.
- Shows the detected character from RuneLite and uses that only to confirm LobsterPot clan access.
- Shows Discord-bot profile data for the logged-in LobsterPot character, including rank, join date,
  point totals, next-rank progress, and pending claims.
- Fetches public broadcasts/events from the built-in GitHub feed URL without requiring login.

## Configuration

No end-user configuration is required.

## Building

Requires a JDK 11 or newer. From the project root:

```sh
./gradlew build
./gradlew run
```

`./gradlew run` launches a developer RuneLite client with the plugin loaded.
